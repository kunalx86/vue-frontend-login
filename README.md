# VUE Frontend for login

## This serves as a boilerplate for a Vue project with Login, Register routes

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
